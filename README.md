# LabDevelop4

This project contains a simple Java application. Below are the steps to clone, compile, and run the project.

## Prerequisites

Ensure you have the following installed on your system:
- Git
- Java JDK (version 11 or higher)

## Steps to Run

1. Clone the repository:
   git clone https://github.com/NazaryiMontytskyi/LabDevelop4

2. Navigate to the project directory:
   cd LabDevelop4/src

3. Compile the main Java file (App.java):
   javac App.java

4. Run the compiled main file:
   java App

## Troubleshooting

- Ensure the JDK is properly installed and the javac and java commands are in your system's PATH.
- Replace https://github.com/NazaryiMontytskyi/LabDevelop4 with the actual URL of the Git repository.
